# Remedy
Remedy is an easy to use medication reminder application

Problem Being Solved

It's easy to forget to take medication on schedule, especially when
managing multiple prescriptions with different times and frequencies.
Missed or mistimed doses can affect treatment effectiveness and, for
some conditions, create real health risks.

Value Provided

Remedy gives users a single, simple place to see what they need to
take and when, with reminders that arrive before a dose is due rather
than after it's missed. It reduces the mental overhead of tracking
multiple medications and helps build a consistent routine.

Features

View today's doses in a clear timeline, with the next upcoming dose highlighted
Add a new medication in seconds (name, dose, schedule)
Mark doses as taken
